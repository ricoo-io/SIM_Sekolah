## Tech Stack & Tools yang Digunakan

### Frontend

- Framework: React (Vite)
- Bahasa: TypeScript
- Styling: Tailwind CSS, Shadcn UI
- Manajemen State / Data: React Query, React Context (Auth)
- Routing: React Router DOM
- Grafik: Recharts
- Ikon: Lucide React
- Formulir: React Hook Form, Zod
- HTTP Client: Axios

### Backend

- Runtime: Node.js
- Framework: Express.js
- Database: MySQL
- ORM: Sequelize
- Otentikasi: JWT (JSON Web Tokens)
- Keamanan: Bcrypt (Hashing Password)
